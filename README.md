## Development

### Requirements

- Rust ([rustup.rs](https://rustup.rs/))
- NodeJS ([nvm.sh](https://nvm.sh))

### Install

```
npm install
cargo install cargo-watch
```

### Watcher

```
npm run dev
```

Visit `http://localhost:3000`
